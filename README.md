# Лабораторная работа по дискретной математике
**Студент:** Гачегова Ирина  
**Группа:** ИТ-12-2025, 1 курс  

## Описание проекта
В данном репозитории содержатся программные реализации алгоритмов для выполнения лабораторной работы "Коды" по дискретной математике. 

В файле main.cpp - решение задачи по подсчету частот встречаемости символов в тексте.

В файле task2.cpp - решение задачи по поиску ключа для зашифрованного текста. Программа перебирает все варианты перестановок и выводит расшифрованный фрагмент текста. 

В файле decoding.cpp - окончательная дешифровка текста. В алгоритм подставлен найденный ключ, что позволило восстановить исходный текст без использования перебора.

**Алгоритм расшифровки**

Программа decoding.cpp работает по принципу блочной перестановки. Текст разбивается на блоки по 6 символов, внутри которых символы переставляются в соответствии с найденным ключом. Это позволяет восстановить исходный порядок букв в словах.
